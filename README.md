# tiny-ui

Tiny 家族产品统一视觉设计系统。

## 引入方式

```html
<link rel="stylesheet" href="assets/root.css">
<link rel="stylesheet" href="assets/reset.css">
<link rel="stylesheet" href="assets/components.css">
```

## 分层职责

| 文件 | 职责 |
|---|---|
| `root.css` | 提供令牌 |
| `reset.css` | 抹平浏览器差异 |
| `components.css` | 组件样式 |
